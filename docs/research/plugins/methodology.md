# Methodology & Scoring

Sources
- GitHub repos (stars, releases, commits), docs, community threads.

Inclusion Criteria
- Open source, free, maintained (12–18 months), clear license, practical value for our flows.

Scoring
- Weighted: Reliability 1.2, Maintenance 1.0, Coverage 1.0, Integration 1.1, Security 1.1, Performance 1.0, UX 1.1. Admission ≥ 3.8.

Limitations
- This offline draft favors widely‑known projects; star/release fields are TBD until refreshed.
