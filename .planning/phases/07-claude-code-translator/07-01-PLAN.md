**8 tests organized in 4 sections:**

**Section: Tool Name Translation (1 test)**

1. **mapToolName translates all 7 entries + passthrough** — Assert:
   - `mapToolName('read') === 'Read'`
   - `mapToolName('write') === 'Write'`
   - `mapToolName('bash') === 'Bash'`
   - `mapToolName('edit') === 'Edit'`
   - `mapToolName('web_search') === 'WebSearch'`
   - `mapToolName('grep') === 'Grep'`
   - `mapToolName('glob') === 'Glob'`
   - `mapToolName('unknown_tool') === 'unknown_tool'` (passthrough)

**Section: YAML Frontmatter Generation (2 tests)**

2. **generateFrontmatter produces correct YAML structure** — Use config with `model: 'claude-sonnet-4-6'`, `tools: ['read', 'write', 'bash']`, `maxTokens: 0`, `prompt: 'do the thing'`. Assert the output:
   - Starts with `---`
   - Contains `model: claude-sonnet-4-6`
   - Contains `allowedTools:` followed by `  - Read`, `  - Write`, `  - Bash` (PascalCase, 2-space indent)
   - Contains `maxTurns: 10` (maxTokens=0 → default 10)
   - Ends with the prompt body after `# Task` header
   - Contains `do the thing`

3. **generateFrontmatter derives maxTurns from maxTokens** — Use config with `maxTokens: 8000`. Assert `maxTurns: 2` (8000/4000=2). Also test `maxTokens: 400000` → `maxTurns: 100` (clamped). Also test `maxTokens: 1000` → `maxTurns: 1` (floor, min 1).

**Section: Temp File Lifecycle (2 tests)**

4. **writeTempAgentFile creates file, cleanupTempFile deletes it** (async) — Write content `'---\nmodel: test\n---\n\n# Task\n\nhello\n'`. Assert:
   - Returned path includes `claude-agent-`
   - Path ends with `.md`
   - `fs.readFile(path)` returns exact content
   - After `cleanupTempFile(path)`, `fs.access(path)` throws (file gone)

5. **Double cleanup is idempotent** (async) — After cleanup in test 4, call `cleanupTempFile(path)` again. Must NOT throw.

**Section: ClaudeAdapter (3 tests)**

6. **Oversized prompt returns failed handle** (async) — Create `ClaudeAdapter`. Build config with `prompt: 'x'.repeat(400_001 * 4)` (>100k token estimate), `id: 'oversize-test'`, `model: 'claude-sonnet-4-6'`, `tools: ['read']`, `maxTokens: 0`, `declaredFiles: []`, `workingDir: '/tmp'`, `timeout: 5000`, `name: 'test'`, `role: 'test'`. Assert:
   - `handle.id === 'oversize-test'`
   - `handle.process === null`
   - `result = await handle.onComplete()` has `status === 'failed'`
   - `result.error` includes `'Prompt too large'`
   - `result.filesCreated.length === 0`
   - `result.tokensUsed.input === 0`
   - `result.runtime === 'claude'`

7. **capabilities() returns correct values** — Create `ClaudeAdapter`. Assert:
   - `caps.sessionResume === true`
   - `caps.jsonOutput === true`
   - `caps.toolPermissions === true`
   - `caps.models.fast === 'claude-haiku-4-5'`
   - `caps.models.standard === 'claude-sonnet-4-6'`
   - `caps.models.reasoning === 'claude-opus-4-6'`
   - `caps.supportedTools` includes `'read'` and `'web_search'`
   - `adapter.runtime === 'claude'`

8. **Factory returns ClaudeAdapter singleton** — Call `clearAdapterCache()` first. Then:
   - `getAdapter('claude')` returns instance of ClaudeAdapter
   - Calling `getAdapter('claude')` again returns the SAME instance (strict equality)
